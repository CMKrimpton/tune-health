Key Don'ts

  • Never use pure black (#000) or pure white (#fff) — use the warm black / white tokens
  • Never hardcode hex values in components — use Tailwind tokens or CSS variables
  • Never use cool grays — always the stone palette
  • Never use transition: all — specify explicit properties
  • Never scale or translate large cards on hover — shadow/glow changes only
  • Never use backdrop-blur heavier than md — kills performance on scroll
  • Never use 100vh — use 100dvh for iOS compatibility
  • Never put text on illustrations
